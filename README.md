# UserManagementJulia

A user authentication API using Julia, Bukdu.jl, HTTP.jl. 

# Run the repo

1. clone the repo 

2. Have the julia 

3. Activate the environment 

          open in the Pkg mode `]`   and `activate .`

4. Instantiate the packages

          instantiate

5. Run the server

          julia --project -i server.jl

6. Stop the server 

           julia --project -i stop_server.jl
